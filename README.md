## 🖥 프로젝트 소개
- 주제 : 숫자야구게임 WEB
- 팀명 : 개인
- 인원 : 1명
- 개발 기간 : 2022.05 ~ 2023.05 (약 1주)


## ⚙ 개발 환경
- Web : jsp
- Server : Java(v1.8), Spring (v4.3.2), Tomcat (v9.0)
- DataBase : PostgreSQL
- Code Editor : STS4, DBeaver


## 🔍 주요 기능
- Login
  - ID, PW 값 DB 검증

- Sign Up
  - ID 중복체크 후 DB 저장

- Game
  - 게임 시작 시 랜덤 문제 생성
  - Thread와 WebSocket을 활용한 타이머
  - 세 자리 답 입력 시 strike, ball, out 내역에 표시
  - 정답을 맞추거나 9회가 지나면 게임 종료 및 랭킹 저장

- Rank
  - 랭킹 확인
